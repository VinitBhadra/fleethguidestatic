==================================================
TROUBLESHOOTING SPECIFIC ISSUES:
==================================================

ISSUE: "Nothing happens when I type"
FIX: 
  - Check browser console (F12)
  - Make sure JavaScript is enabled
  - Try hard refresh (Ctrl+Shift+R)

ISSUE: "Only some calculators work"
FIX:
  - Make sure you're typing in ALL required fields
  - Fuel calc needs: distance + price + mileage (3 fields)
  - Trip calc needs: distance + price + mileage (3 required, driver & toll optional)
  - Cost/KM needs: total + distance (2 fields)

ISSUE: "Results flash and disappear"
FIX:
  - Don't use negative numbers
  - Don't use zero
  - Make sure all required fields have values

ISSUE: "JavaScript error in console"
FIX:
  - Clear browser cache
  - Re-extract files from tar.gz
  - Make sure main.js file is present in js/ folder
